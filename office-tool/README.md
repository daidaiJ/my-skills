# 办公工具

文档处理全家桶，支持 Word、PDF、PowerPoint、Excel 等格式。

| Skill | 用途 |
|-------|------|
| [docx](./docx/) | Word 文档创建、编辑、格式化 |
| [pdf](./pdf/) | PDF 读取、合并、拆分、表单填写、OCR |
| [pptx](./pptx/) | PowerPoint 演示文稿创建和编辑 |
| [xlsx](./xlsx/) | 电子表格处理，支持公式、图表、数据清洗 |
| [liteparse](./liteparse/) | 轻量 PDF 文本提取，本地处理无依赖 |
