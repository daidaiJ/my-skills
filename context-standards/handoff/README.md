# Handoff

会话交接工具，将当前对话压缩为交接文档，供新 agent 继续工作。

## 功能

- 生成结构化的交接文档，保存到用户临时目录
- 包含 "推荐 Skill" 章节，建议后续 agent 应调用的技能
- 引用而非重复已有文档（PRD、计划、ADR 等）
- 自动脱敏（API key、密码、个人信息）

## 使用方式

```
/handoff 描述下一次会话的重点
```

参数作为对后续会话的指引，文档会据此调整内容侧重。

## 适用场景

- 长对话需要切换新会话继续
- 将工作交给其他 agent 接手
- 保存当前进度以便后续恢复

> 详细用法参见 [SKILL.md](./SKILL.md)
